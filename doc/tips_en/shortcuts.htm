F2 Calibration

Ctrl + Enter AutoSet
Ctrl + 0 Reset
Ctrl + S Save wave
1/2 Turn on/off Ch1/Ch2
Tab Select next channel

Q/A Change Ch1 Voltage scale

Left/Right Move trigger position
BackSpace Set trigger position to the beginning
